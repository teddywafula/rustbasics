# Function
- Basic building blocks for reusable code
 making it easier to read and maintain.

## Function Definitions

### Define a function

fn function_name(params) {
    // function body
}

### Invoke a function

fn function_name (params) {
}

fn main() {
    function_name(params);
}

### Function with return type
fn function_name (params) -> return_type {
    return value
}

fn function_name (params) -> return_type {
    value
}
